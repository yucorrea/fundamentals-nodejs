## :information_source: How to use

### Install API

```bash
# Clone the repository
$ git clone https://github.com/yucorrea/fundamentals-nodejs

# Navigating to the backend folder
$ cd nodejs-concepts

# Install dependecies
$ yarn install

# Start application
$ yarn dev:server

# Run tests
$ yarn test

# Server is running
```
